Welcome to the Helm Charts Repository

This repository contains Helm charts for deploying applications in your Kubernetes environment.

Available Charts:

  • Maze Chart: Helm chart for deploying Maze application.

To install the Maze chart, use the following Helm command:

    helm repo add maze-repo https://github.ibm.com/MAZE-MultiCloud/helm-charts/gh-pages/
    helm repo update
    helm install maze maze-repo/maze

For more information, visit the official documentation of the charts.
